jQuery CSS Methods Demo

css(propertyName): Retrieves the current computed value of a CSS property for the first element in the set of matched elements.

css(propertyName, value): Sets one or more CSS properties for the set of matched elements.

css(propertiesObject): Sets multiple CSS properties for the set of matched elements using an object.

addClass(className): Adds one or more classes to each element in the set of matched elements.

removeClass(className): Removes one or more classes from each element in the set of matched elements.

toggleClass(className): Toggles a class on and off for each element in the set of matched elements.

hasClass(className): Checks if at least one of the elements in the set has the specified class.
